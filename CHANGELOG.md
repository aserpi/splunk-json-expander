# Changelog

## 2.1.0 2026-06-04

Independently configure how many levels deep the JSON should be expanded for automatic and manual triggers.

## 2.0.0 - 2026-05-30

Improvements:

- Leave the `Max expansion level` option empty to expand all levels.

Bug fixes:

- On icon click, expand only the current search.

Add an option to automatically expand JSON search results on page load.

Initial release.
